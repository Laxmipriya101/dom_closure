Q3. 
        for (var i = 0; i < 3; i++) {
          setTimeout(function log() {
            console.log(i); // What is logged?
          }, 1000);
        }

Output: 3 3 3

Explanation: Here the setTimeout takes the latest value , In this condition the latest value is 3 . For traversing its print 3 times 3 value .
